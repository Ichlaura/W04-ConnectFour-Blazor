# Connect Four - Blazor App

This project is a Connect Four game built using .NET Blazor as part of the CSE 325 course at BYU–Idaho (2026 Block 2).

The application is based on concepts from Microsoft Learn and demonstrates interactive web development using C# instead of JavaScript.

## 🎯 Features

- Interactive Connect Four game
- Turn-based gameplay (Player 1 vs Player 2)
- Automatic win detection (horizontal, vertical, diagonal)
- Draw detection
- Move history tracking
- Smart column blocking when full
- Clean and responsive UI

## 🛠️ Technologies Used

- .NET 8 / .NET 10
- Blazor Server
- C#
- Razor Components
- CSS
